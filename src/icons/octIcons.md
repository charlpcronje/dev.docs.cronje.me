Octicons are a set of SVG icons built by GitHub for GitHub

## JavaScript

The octicons node.js library is the main JavaScript library. With [a JavaScript API](https://github.com/primer/octicons/blob/main/lib/octicons_node/README.md) that can be used in a variety of applications.
